# MealPalAssignment

## Starting the NextJS React App

1.

```bash
 cd front-end
 ```

2.

```bash
 npm install
 ```

3.

```bash
 npm run dev
 ```

4. Navigate to localhost:3000 from your browser

## Starting the Express Backend Server

1.

```bash
 cd back-end
 ```

2.

```bash
 npm install
 ```

3.

```bash
 node server.js
 ```
